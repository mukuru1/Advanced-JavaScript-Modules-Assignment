# My JavaScript Project

This project implements advanced JavaScript modules, including CommonJS and ES6 modules, featuring a scientific calculator, a temperature converter, and a search function through an array of student objects.

- [Installation](#installation)
- [Usage](#usage)
- [Features](#features)

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/my-js-project.git
   ```
2. Navigate to the project directory:
   ```
   cd my-js-project
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To run the project, execute the following command:
```
node src/index.js
```

This will demonstrate the functionalities of the scientific calculator, temperature converter, and search functions.

## Features

- **Scientific Calculator**: Performs basic arithmetic operations and logs results.
- **Temperature Converter**: Converts Celsius to Fahrenheit and provides temperature conditions (Hot, Warm, Cold).
- **Search Functionality**: Allows searching through an array of student objects by ID and name, and filtering by age.
